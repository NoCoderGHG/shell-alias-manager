# Bash Alias Manager

GTK3 tool to manage bash aliases in `~/.bashrc`.

## Requirements

```bash
sudo apt install python3-gi gir1.2-gtk-3.0
```

## Run

```bash
python3 alias_manager.py
```

## Features

- View, add, edit, and delete bash aliases
- Saves directly to `~/.bashrc` under a managed block
- Does not touch other content in `~/.bashrc`
- i18n: Deutsch / English / System (auto-detect)
- Config: `~/.config/alias-manager/config.json`
